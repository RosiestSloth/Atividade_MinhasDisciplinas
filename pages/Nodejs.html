Node.js

O que é Node.js?

Node.js é uma plataforma construída sobre o motor V8 do Google Chrome que permite executar código JavaScript no servidor. Ele é amplamente utilizado para criar aplicações web escaláveis, APIs e sistemas em tempo real.

História e Evolução

Criado em 2009 por Ryan Dahl, o Node.js revolucionou o desenvolvimento backend ao permitir que desenvolvedores utilizassem JavaScript fora do navegador. Desde então, tornou-se uma das principais tecnologias para servidores e aplicações modernas.

Principais Características

  • Baseado em eventos e orientado a assincronicidade
  • Alta performance graças ao motor V8
  • Grande ecossistema de módulos via npm
  • Utiliza JavaScript no backend
  • Ideal para aplicações em tempo real

Exemplo Básico

Veja um exemplo simples de servidor HTTP com Node.js:


const http = require('http');
const hostname = '127.0.0.1';
const port = 3000;

const server = http.createServer((req, res) => {
  res.statusCode = 200;
  res.setHeader('Content-Type', 'text/plain');
  res.end('Olá, este é um servidor Node.js!');
});

server.listen(port, hostname, () => {
  console.log(`Servidor rodando em http://${hostname}:${port}/`);
});

Onde o Node.js é Utilizado?

  • APIs e servidores web
  • Aplicações em tempo real (chats, jogos, etc.)
  • Ferramentas de automação e CLI
  • Microserviços
  • Internet das Coisas (IoT)

Principais Frameworks e Bibliotecas

  • Express – Framework minimalista para servidores web
  • NestJS – Framework para aplicações escaláveis
  • Socket.io – Comunicação em tempo real
  • Koa – Framework moderno e leve
  • PM2 – Gerenciamento de processos Node.js

Curiosidades

  • Node.js utiliza um modelo de I/O não bloqueante, ideal para aplicações escaláveis.
  • O npm, gerenciador de pacotes do Node.js, é um dos maiores repositórios de software do mundo.
  • Grandes empresas como Netflix, LinkedIn e Uber utilizam Node.js em suas aplicações.
